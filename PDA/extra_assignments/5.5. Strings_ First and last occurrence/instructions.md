# Statement

Given a string that may contain a letter `f`. Print the index of the first and last occurrence of `f`. If the letter `f` occurs only once, then output its index once. If the letter `f` does not occur, print -1.

# Example input #1

```
comfort
```

# Example output #1

```
3
```

# Example input #2

```
office
```

# Example output #2

```
1 2
```

# Example input #3

```
hello
```

# Example output #3

```
-1
```
