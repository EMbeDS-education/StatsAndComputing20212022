# Statement

Let's call an integer a _palindrome_ if it remains the same after reading its digits from right to left. Given a four-digit integer, print "YES" if it's a palindrome and print "NO" otherwise. 

# Example input #1

```
1221
```

# Example output #1

```
YES
```

# Example input #2

```
1234
```

# Example output #2

```
NO
```
