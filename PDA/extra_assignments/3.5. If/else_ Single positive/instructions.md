# Statement

Given two non-zero integers, print "YES" if exactly one of them is positive and print "NO" otherwise.

# Example input #1

```
-5
```

```
10
```

# Example output #1

```
YES
```

# Example input #2

```
5
```

```
10
```

# Example output #2

```
NO
```
