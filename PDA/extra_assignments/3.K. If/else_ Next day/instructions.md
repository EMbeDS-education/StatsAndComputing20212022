# Statement

Given a month (an integer from 1 to 12) and a day in it (an integer from 1 to 31) in the year 2017, print the month and the day of the next day to it.

# Example input #1

```
3
```

```
30
```

(March 30)

# Example output #1

```
3
```

```
31
```

# Example input #2

```
3
```

```
31
```

(March 31)

# Example output #2

```
4
```

```
1
```
