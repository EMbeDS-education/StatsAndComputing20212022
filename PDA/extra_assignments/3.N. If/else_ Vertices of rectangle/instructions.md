# Statement

Given integer coordinates of three vertices of a rectangle whose sides are parallel to coordinate axes, find the coordinates of the fourth vertex of the rectangle.  


 

# Example input #1

```
1
```

```
5
```

```
7
```

```
5
```

```
1
```

```
10
```

three vertices are (1, 5), (7, 5), (1, 10)

# Example output #1

```
7
```

```
10
```
